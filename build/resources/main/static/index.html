DNA VCF to TXT Converter

A simple Spring Boot application that converts raw DNA VCF files into TXT format compatible with genealogy tools.

What does this app do?

This tool reads a VCF file, parses genotype records, converts them into the required TXT output format, and downloads the result automatically to your default browser downloads folder.

Upload your VCF file

Supported format: .vcf | Maximum file size: 50 MB
After successful conversion, the TXT file will be downloaded automatically.
